Activity 1

What is an Iot Application

  • Iot Stands for Internet of things
  • Applications for the Internet of Things (IoT) are devices and systems that connect to the internet and share data to automate various processes and operations.

What are the Main Features Needed???

  • Communication between Multiple Devices
  • Storage of Data
  • Real-time monitoring
  • Integration
  • Scalability
  • Automation

What are some examples?

  • Smart Locking System
  • Smart Home
  • Smart Shopping Cart
  • Smart eatery and Evacuation System
  • Smart Wardrobe System
  • Security Systems
  • Fitness trackers
  • supply chain management
  • remote patient monitoring
  • Smart Hospitals
  • smart traffic management
  • Livestock Monitoring

Activity 2

What does smart means?

  • Building equipped with advanced and modern technology
  • Optimizes performance, such as energy efficiency, comfort and safety
  • Uses data and automation systems
  • Combines IoT devices and sensors for live monitoring and analysis
  • Offers sustainability

What are the main features??

  • Connectivity
  • Data Security
  • Integeration with other system
  • Ability to analyze and interpret collected data to make informed decisions.
  • Data Collections

What are some smart building scenerios??

  • Management
  • Access control
  • Environmental Monitoring
  • Maintenance Management
  • Asset tracking
  • Emergency Response
  • Meeting Room Management
  • Occupant Comfort

What are the high level features???

  1. Using ultrasonic sensors or PIR sensors to sense motion and act on command to save assets like electricity
  2. Security Features including Fire Detection
  3. Using smoke or gas sensors for safety purposes in kitchen area.
  4. Using light intensity sensors for automation purposes.
  5. Using Fingerprint sensors for security purposes.

What are the parts needed???

  1. Screen for the user to interact with the systems
  2. Monitoring devices : such as sensors like HC-SR04
  3. keyboard and mouse to interact with the interface
  4. Security Features including Fire Detection using MQ2
  5. Using ultrasonic sensors or PIR sensors to sense motion and act on command to save assets like electricity
